### User Command Usage

To list 10 users:

``` bash
im user list -l=10 
```

To create a user:

``` bash
im user create username password
```

To delete a user:

``` bash
im user delete username
```

To reset a user's password:

``` bash
im user reset-password username password
```

To get user's info:
``` bash
im user get username
```
